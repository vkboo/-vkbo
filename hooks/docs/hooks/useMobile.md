# useMobile

## API
```typescript
const isMobile = useMobile();
```

### Result
| Property            | Description                                                                                      | Type                                         |
| ------------------- | ------------------------------------------------------------------------------------------------ | -------------------------------------------- |
| isMobile            | Is it currently a mobile mode?                                                                   | `boolean`                                    |
